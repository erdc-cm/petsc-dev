DOE 2000 ACTS Numerics Toolkit Projects
Equation Solver Interface (ESI)
DOE2000 1999 PI Meeting Status Report

  1. Project Title: Equation Solver Interface (ESI)

  2. Project Homepages: http://z.ca.sandia.gov/esi

  3. Objective: Define a set of interfaces for the scalable solution of sparse linear systems arising from scientific simulations and related problems. These will

    (1) be usable from Fortran, C, and C++,
    (2) be extensible, so that additional interfaces may be added in the future without breaking previously developed ESI code
    (3) when possible, leverage off already existing object oriented or object based solver libraries.

  4. Approach: Hold quarterly meetings to discuss the standard design and its associated software engineering issues. In addition we have an active mailing list for technical discussions. Develop implementations that test the proposed approaches. The technical design approach is object oriented.

  5. Recent FY98 accomplishments: We've had four meetings in the past year and released two iterations of the specification.

  6. Plans for FY99: At least three groups (1) the ISIS++ development team (SNL) , (2) the PETSc development team (ANL) and (3) the Hypre preconditioner group (LLNL) are planning to provide full support for the ESI, this means that the ISISI++ and PETSc solvers and Hypre preconditioners will be usable through the ESI interface. We will be having regular conference call meetings, using Microsoft Netmeeting to share a common desktop, to iron out the exact details of the specification's header files. These will complement the already productive email discussions.

  7. Tool availability: The current draft specifications are available at the home page. These will be updated as the specification develops. The ISIS++, PETSc and Hypre   implementations will be made available as they develop for external testing.
