DOE2000 1999 PI Meeting Status Report

The PETSc group is directly involved in the following 7 ACTS Numerical toolkit projects

  • PVODE-PETSc interface (completed)
  • Aztec-PETSc interface (nearing completion)
  • Equation solver interface (ESI)
  • Application toolkit promotion and support
  • Common component architecture (CCA)
  • Numerical Optimization (including OT - the Optimization Toolkit)
