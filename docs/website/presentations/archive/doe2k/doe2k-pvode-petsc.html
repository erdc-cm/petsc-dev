DOE 2000 ACTS Numerics Toolkit Projects
PETSc-PVODE Project
DOE2000 1999 PI Meeting Status Report

  1. Project Title: PVODE-PETSc Interface

  2. Project Homepages: http://www.mcs.anl.gov/petsc
                                    http://www.llnl.gov/CASC/PVODE

  3. Objective:

    (1) Allow application developers to seamlessly use the sophisticated stiff ODE integrators in PVODE with the parallel matrices and preconditioners provided in PETSc.

    (2) Provide efficient parallel computation of sparse Jacobians required for constructing preconditioners using graph coloring and finite differences.

  4. Approach:

    5. (1) Since both PVODE and PETSc are C object-based libraries, we developed object adaptors (i.e. object wrappers) to make PETSc object implementations coexist with PVODE objects. For example, PETSc vectors and PVODE vectors provide different interfaces to the same basic functionality; object adaptors simply map these interfaces to interfaces that the other package expects.

    (2) Using the existing parallel communication constructs in PETSc, along with newly developing coloring code we implemented a new component to efficiently compute the Jacobians for specific sparse matrix formats.

  5. Recent FY98 accomplishments: The interface has been completed and tested. These tools were tested on both model problems and a specific application that previously used LSODE (a previous generation of the PVODE author's work) that required more scalable integrators. The application is microstructural evolution in sintering and involves the simulations of the dynamics of molecular interactions via the method of lines.

  6. Plans for FY99: This project has been completed.

  7. Tool availability: PETSc is fully available from its web-page and supported via petsc-maint@mcs.anl.gov. It is also installed on the NERSC system. PVODE  is currently available via special request and will be publicly available shortly.
