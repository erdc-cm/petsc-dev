Simple Output from a PETSc Program

 

In parallel programs one usually wishes to either

  • print a single message independent of number of processors

    PetscPrintf()

     

  • each processor prints its own message

    PetscSynchronizedPrintf()

    PetscSynchronizedFlush()

${PETSC_DIR}/src/sys/examples/tutorials/ex2.c demonstrates the use of these routines (which you just compiled and ran).

${PETSC_DIR}/src/sys/examples/tutorials/makefile used for compiling the example
