DOE 2000 ACTS Numerics Toolkit Projects
DOE2000 ACTS Numerics Toolkit Use Project
DOE2000 1999 PI Meeting Status Report

  1. Project Title: ACTS Numerics Toolkit Use

  2. Project Homepage: http://acts.nersc.gov

  3. Objective: Promote the use of the ACTS Toolkits in ASCI, DP, ER, and non-DOE application codes. Incorporate feedback from toolkit users to develop additional enhancements and optimizations for the toolkits. Assist users in using two or more toolkits in the same application. The project is related to, but distinct from the NERSC support effort in that here the toolkit developers interact directly with the end large-scale application users.

  4. Approach: Develop personal contacts with select application developer groups; provide support for additional users via e-mail maintenance lists. Advertise the toolkits via NERSC, Netlib, NHSE, and news groups.

  5. Recent FY98 accomplishments: The toolkits have over 500 users at the various DOE laboratories, universities, and industry. Sample application performance results for the various toolkits.

  6. Plans for FY99: Continue active support, including recruiting additional users representive of different types of simulations. Toolkit developers have daily interaction with end users.

  7. Tool availability: The NERSC web-pages, http://acts.nersc.gov, have a current list of tools and availability status.
