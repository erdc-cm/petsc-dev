DOE 2000 ACTS Numerics Toolkit Projects
Aztec-PETSc Project
DOE2000 1999 PI Meeting Status Report

  1. Project Title: Aztec-PETSc Interface

  2. Project Homepages: http://www.mcs.anl.gov/petsc
                                    http://www.cs.sandia.gov/CRF/aztec1.html

  3. Objective: Allow current and prospective Aztec users to harness some of the linear solver, error handling and profiling technology in the PETSc package and allow PETSc users to use the matrix storage format and preconditioners available in Aztec.

  4. Approach:

    (1) Provide PETSc object adaptors (object wrappers) for the Aztec matrix and preconditioner formats (this involves use of the PETSc MatCreateShell() and PCCreateShell() constructs).  This allows the Aztec objects to be used as PETSc objects in, for example,  the PETSc Krylov solvers or even nonlinear solvers.

    (2) Provide an Aztec main solver interface that calls the underlying PETSc solvers. This allows Aztec users to test some PETSc functionality without having to heavily modify their application code.

  5. Recent FY98 accomplishments: The interface has been completely designed and the implementation is nearly complete. Basic testing has begun.

  6. Plans for FY99: We plan to complete the implementation and basic testing and then test with a real application code.

  7. Tool availability: PETSc and Aztec are both fully available from their web-pages and installed on the NERSC system. We hope to make the interface adaptors publicly available in the next few months.
