 We are now developing a major expansion of PETSc to provide higher level interfaces (equations, meshes and discretizations) for solving PDEs. This package will complement (and, of course, use) the current PETSc algebraic solver software. In order to prevent confusion, the current PETSc software, which provides scalable algebraic solvers, will be known as PETSc-AS while the new software will be called PETSc-CS (for continuous solvers). Both together will be known as PETSc.

Current users of PETSc will be able to use PETSc-CS features, but they can continue to only use the current PETSc algebraic solvers, now renamed PETSc-AS, if they desire.

PETSc-CS used to be refered to, by some, as PETSc 3; the PETSc 3 name was dropped because it was too confusing.
